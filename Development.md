# Development

Install dependencies

```bash
$ bun install
```

Build

```bash
$ brew install zig
$ brew install go
$ brew install --build-from-source upx
$ go install github.com/dosgo/zigtool/zigcc@latest
$ go install github.com/dosgo/zigtool/zigcpp@latest
$ bun run build
```

Run example

```bash
$ bun ./example.ts
```
